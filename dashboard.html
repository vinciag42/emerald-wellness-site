Prior Lab Upload + Trend Review

Best place for previous labs: add the file here, choose what you want reviewed, enter lab numbers manually, then save notes for quarterly comparison. To enter biological-age values, choose each marker, enter its value, unit, date, and flag, then select Add Lab Value.

Biological Age Inputs

Use this as a premium lab organizer for phenotypic-age style review. First save the input set and chronological age, then enter each lab value in the manual lab-entry box.

9-marker values to enter

Albumin, alkaline phosphatase, creatinine, glucose, C-reactive protein/hs-CRP, lymphocyte %, MCV, RDW, and WBC. Enter one marker at a time using the manual lab-entry fields.
